Задание 1

Используя графический редактор (Adode Photoshop, Figma, Paint) или ручку с бумагой, нарисуйте структуру сайта (шапка, меню, контент, подвал) с объектами (текст, поле ввода, кнопки, картинки и др.). Подпишите отдельно какие языки за какую часть будут отвечать и как.

Задание 2

Почему стоит разделять некоторые языки программирования при разработке программ, приложений для операционных систем IOS/Android? В чём преимущество и недостаток каждой из ОС, которые упрощают ил усложняют процесс разработки?

Задание 3

Предложите идею приложения на компьютер. Какой язык программирования наиболее для этого подходит?

Задание 4

Опишите идею игры, которую вы хотите реализовать и процесс его создания. Выберите подходящий движок с языком программирования, объяснив выбор.

Задание 5

Приведите 2-3 примера задач машинного обучения (распознавание изображений, анализ текста). Поделитесь мнением касательного того, почему Python или Java подходит для машинного обучения и развития ИИ.
