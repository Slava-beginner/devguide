HTML

HTML (Hypertext Markup Language) – это язык разметки, который используется для создания структуры и содержания веб-страниц. Но сам по себе HTML не может обеспечить полноту веб-приложения, поэтому используются другие технологии:

Возможности HTML:

  • Делать текстовую разметку — форматировать текст, выделять фрагменты, создавать списки, добавлять сноски.
  • Встраивать медиа. HTML позволяет размещать на сайте изображения, аудио, видео, карты.
  • Создавать ссылки и навигацию. Гиперссылки и списки меню помогают быстрее найти информацию и сориентироваться на странице.
  • Создавать формы. Формы нужны для регистрации посетителей сайта по телефону и электронной почте, оформления заказов, опросов и сбора обратной связи — отзывов, комментариев, предложений.

В HTML можно даже создавать простой дизайн: например, устанавливать цвет и шрифт текста или фоновый цвет блока. Но более сложный дизайн страницы разработчики делают с помощью CSS — языка стилей, который создали специально для работы в связке с HTML.

Возможностей HTML не хватает, чтобы «оживлять» сайты, делать их функциональными и интерактивными. Здесь приходит на помощь язык программирования JavaScript, благодаря которому мы можем взаимодействовать с формами на сайте, видеть динамические элементы и анимации.

Как выглядит код на HTML

Увидеть HTML-код очень просто. Для этого нажмите клавишу F12 на любом сайте или правую кнопку мыши и выберите «Показать исходный код» или «Просмотр кода страницы». В разных браузерах названия этой команды может отличаться.

Внизу приедствален типовой код HTML:

HTML-код пишут и сохраняют в документе — текстовом файле с расширением .html или .htm. HTML-файл можно написать в любом текстовом редакторе, даже в стандартном Блокноте Windows. Чтобы просматривать HTML-файлы, интернет не нужен. Если создать несколько HTML-файлов в одной папке и поставить внутри гиперссылки, то можно переходить по этим ссылкам из одного документа в другой.

Теги HTML

Теги HTML — это команды, которые говорят браузеру, что и в каком порядке показывать на экране. У каждого тега есть имя, которое расположено в угловых скобках. Часто используются следующие теги:

<head> в этом теге хранится служебная информация — заголовок и кодировка страницы

Внутри тега <body> помещается содержимое страницы, которое отображается в браузере — текст, изображения, видео.

В теге <title> — заголовок веб-страницы. Его браузер отобразит как название и сохранит в описании, если сохранить страницу в закладки.

В тегах <h1-h6, p> пишется текст,который отображается на странице. Вервые теги вставляют загаловки,а последний - основной текст

Ненумерованные или маркированные списки («буллеты») добавляют на страницу тегом <ui>. Для создания нумерованного списка используется тег <oi>. Отдельный элемент в списке любого типа вводится тегом <li>

Тег <script> может содержать JavaScript-код или ссылаться на внешний файл .js

Тег <img> добавляет изображения

Тег <link> ссылвется на внешние файлы, например .css, .ico и т.д

CSS

CSS (Cascading Style Sheets) — это язык, который используется для стилизации и оформления веб-страниц. Он является ключевым компонентом веб-разработки. CSS отвечает за визуальное оформление страниц: цвет, шрифты, отступы, выравнивание, анимации и многое другое.


С чем используется css?

CSS тесно связан с HTML и JavaScript. HTML задаёт структуру веб-страницы (заголовки, текст, изображения и т.д.). CSS применяется к элементам HTML, чтобы задать их стиль.
JavaScript используется вместе с CSS для редактирования стилей с применением интерактивности и интерактивности.

Назначение CSS

CSS служит для управления внешним видом и восприятием HTML-страницы, включая:

  • Цвета текста, фонов и других элементов
  • Типографику: шрифты, размер текста, межстрочное расстояние и выравнивание
  • Макет страницы: отступы, поля, позиционирование и сетки;
  • Эффекты: переходы, анимации, тени и фильтры.
  • Анимации без использования JavaScript
  • Адаптивность: создание адаптивного дизайна для изменения отображения в зависимости от размеров устройств, на котором просатривается сайт

Особенности работы CSS

CSS работает по принципам, которые помогают лучше управлять стилями на странице.
CSS может складываться слоями: сначала браузер смотрит, какой стиль указан в одном месте, а потом добавляет стиль из другого, если он не противоречит первому. Это помогает использовать общие стили, а потом уточнять детали в нужных местах.
Некоторые свойства автоматически применяются к "дочерним" элементам. Например, если вы задаёте цвет текста для всей страницы, то этот цвет автоматически применяется ко всем заголовкам и абзацам на ней.
Если есть два разных стиля для одного и того же элемента, CSS решает, какой стиль важнее. Например, если для текста указаны два разных цвета, то приоритет будет у того стиля, который более "специфичен" (точен). Это как если бы один стиль был общим для всех абзацев, а второй указан для конкретного абзаца.

Как устроен CSS

CSS состояит из стилей. Стили — это визуальные особенности, которые нужно придать странице, например цвет текста, размер подзаголовка, тип шрифта или оформление иллюстрации.

Стили или свойства CSS называются правилами. Каждое правило состоит как минимум из одного селектора и одной пары «свойство — значение».

Селектор нужен для того, чтобы браузер понимал, к какому элементу CSS будет применяться свойство: заголовку, абзацу, фоновому цвету.

Селектор нужен для того, чтобы браузер понимал, к какому элементу CSS будет применяться свойство: заголовку, абзацу, фоновому цвету. Свойство показывает, что именно будет меняться в этом элементе: цвет, размер, выравнивание. Значение определяет, как именно изменится элемент: станет красным, увеличится вдвое, примет круглую форму.

JavaScript

JavaScript (JS) — язык программирования высокого уровня, используемый для создания интерактивных элементов на веб-страницах. Данный язык работает на основе скрипта - это документ, описывающий действия, необходимые для обработки и выполнения. Работать со скриптами сможет каждый браузер.


Назначение JavaScript

JavaScript используется в веб-разработке для взаимодействия с пользователем и "оживления" страниц сайта, делая их красочными, динамическими.

  • Интреактивность веб-страниц. Показ всплывающих сообщений, реакция на дейчтвия пользователя.
  • Динамическое обновление содержимого. Изменение содержимого веб-страницы.
  • Создание анимации и визуальных эффектов
  • Работа с формами. Проверка ввёдённых данных,вводимых пользователем
  • Сохранение данных в браузере.
  • Блокчейн. Используется для разработки клиентских приложений, взаимодействующих с блокчейн-технологиями.

JS добавляет анимацию и всплывающие уведомления, валидирует формы, обновляет контент и заставляет интерфейс реагировать на наши действия. Например, наводишь курсор на кнопку, а она меняет цвет (или исчезает после нажатия). В общем, язык JavaScript делает страницы функциональными и отзывчивыми.

Особенности работы JS

Интерпретируемость. JavaScript выполняется в реальном времени в среде браузера, без необходимости предварительной компиляции. Разработчик вносит изменения в код JavaScript и сразу видит результат. Это удобно.

Интеграция с HTML и CSS. В связке с ними язык JavaScript позволяет создавать полнофункциональные веб-страницы.

Динамическая типизация. Тип переменных в JavaScript определяется автоматически при выполнении кода. Программистам не нужно присваивать его заранее, что упрощает работу.

Большое количество библиотек и фреймворков JavaScript. Готовые решения для различных задач экономят время IT-специалистов.

Кроссплатформенность. Код JavaScript может быть выполнен на разных платформах и устройствах.

Автоматический сбор мусора. Облегчает управление памятью, автоматически удаляя объекты, которые не используются.

Функции и операторы условий

Переменные служат для хранения и управления данными в программе. В JavaScript переменные объявляются с помощью ключевых слов var, let или const.

Арифметические операторы — это операторы, которые выполняют математические операции, такие как сложение, вычитание, умножение и деление.

Операторы сравнения — это операторы, которые сравнивают два значения и возвращают логическое значение (true или false).

Логические операторы — это операторы, которые соединяют или инвертируют логические значения.

Побитовые операторы — это операторы, которые работают с битами (единицами и нулями) значений.

Операторы присваивания — это операторы, которые присваивают значение переменной или свойству объекта.

Функции -это "подпрограмма" которая выполняет какое-либо действие и содержит последовательность. Операторы (f, else) содержат установки по выполнению какой либо операций исходя из условий. Точка с запятой — это символ, который используется для завершения инструкций. Он показывают, где заканчивается одна инструкция и начинается другая.

SQL

SQL (Structured Query Language) — язык для управления и взаимодействия с базами данных. Данный язык программирования нужен для хранения и обработки информации в базе данных. Инструкции SQL можно использовать для хранения, обновления, удаления, поиска и извлечения информации из базы данных. Можно также использовать SQL для поддержания и оптимизации производительности базы данных. По своему виду код SQL понятен для человека.

База данных — это способ хранения и организации данных, которые структурированы так, чтобы пользоваться ими могли и человек, и алгоритм. При помощи SQL можно работать с реляционными базами данных — то есть теми, где данные представлены в виде таблиц. Это отчасти похоже на таблицы в Excel, при этом все таблицы в рамках одной базы данных связаны между собой сквозным поиском.

Назначение SQL

Главные задачи SQL — составлять запросы так, чтобы находить среди большого объёма информации ту, что нужна для конкретных целей, сортировать её, структурировать и представлять в наиболее простом и понятном виде. Данный язык запросов предназначен для:

  • Собирать и хранить данные в виде таблиц
  • Изменять их содержимое и структуру
  • Объединять данные и выполнять вычисления
  • Защищать и распределять доступ

Особенности SQL

  • Это язык запросов, а не программирования. Его используют в дополнение к Python, JavaScript или C++, но только для работы с базами данных. Написать на нём полноценный сайт или приложение невозможно.
  • Чёткая и понятная структура. Это делает язык SQL для работы с данными относительно простым для начинающих.
  • Универсальность. Есть единые стандарты построения запросов для любых баз данных и браузеров, которые позволяют обрабатывать даже очень большие объёмы информации.
  • Совместный доступ. SQL позволяет создавать интерактивные запросы. Это значит, что можно получать нужные данные онлайн и принимать решения на их основе.
  • Управление доступом. При помощи SQL можно предоставить, закрыть или ограничить доступ к данным для разных групп пользователей, а также выдать им определённый набор функций: чтение, изменение, создание, удаление, копирование. Это защищает базы данных от взлома или несогласованных изменений.

    Объявление переменной определяет тип данных переменной и, при необходимости, инициализирует ее значением

    Арифметические операторы выполняют основные математические операции, такие как сложение, вычитание, умножение, деление и умножение по модулю

    Реляционные операторы используются для сравнения двух значений. Они возвращают логическое значение (true или false) на основе сравнения

    Логические операторы используются для выполнения логических операций над логическими значениями. Логическими операторами в C++ являются AND (&&), OR (||), а НЕ (!)

    Условные операторы позволяют программе выполнять определенные блоки кода в соответствии с заданными условиями. Для этой цели обычно используются операторы if, else if и else else

    Оператор switch используется для выбора одного из множества блоков кода для выполнения. Он вычисляет выражение и выполняет код на основе соответствующей метки case

    Циклы используются для многократного выполнения блока кода до тех пор, пока не будет выполнено заданное условие. C++ поддерживает три типа циклов: for, while и do-while

    Инструкция break используется для преждевременного завершения цикла, в то время как инструкция continue используется для пропуска оставшегося кода в цикле и перехода к следующей итерации

    В C++ функции объявляются с использованием типа возвращаемого значения, имени функции и необязательных параметров. Определение функции содержит реализацию логики работы функции

    При объявлении и инициализации массивов в C++ программисты указывают тип данных элементов, которые будет содержать массив, и определяют его размер в квадратных скобках. Этот процесс выделяет смежные области памяти для хранения элементов указанного типа, обеспечивая эффективный поиск данных и манипулирование ими.

    Классы – это определяемые пользователем типы данных, которые объединяют данные и функции в единое целое. Объекты – это экземпляры классов

    Конструкторы инициализируют объекты, в то время как деструкторы очищают ресурсы при уничтожении объектов. Это специальные функции-члены с тем же именем, что и у класса

    Инкапсуляция – это объединение данных и функций в единый модуль (класс). Наследование позволяет классам наследовать свойства и поведение от других классов. Полиморфизм позволяет объектам принимать различные формы в зависимости от их контекста

    Блок try используется для включения кода, который может генерировать исключение. Блок catch перехватывает и обрабатывает исключения, генерируемые в блоке try. Оператор throw создает исключение явно

    Обработка исключений обеспечивает плавное восстановление после ошибок, предоставляя механизм для перехвата и обработки исключений. Это предотвращает сбой программы и обеспечивает контролируемую обработку ошибок.
